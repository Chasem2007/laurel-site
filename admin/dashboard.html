Manage Accounts
ℹ️ How Admin Accounts Work

Create a login for any staff member who needs to post announcements. Steps:

  1. Click + Create Admin Account above
  2. Enter a username (e.g. msmith) and a temporary password
  3. Optionally check Can Create Users to let them also create accounts
  4. Share the credentials with that staff member — they log in at /admin

Admins can only see and manage their own posts. They can change their own username and password from the "My Account" page.
